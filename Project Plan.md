# 1. Project Overview

- **프로젝트 이름**: TagNamer
- **프로젝트 주제**: 수십~수천 개의 파일을 일괄적으로 이름 변경하는 Windows 프로그램
- **개발 환경**
  - **OS**: Windows 11
  - **IDE**: VSCode
  - **AI Tools**: Gemini Web, Gemini Code Assist
  - **Language**: C#
- **개발 목적**: 기존에 사용하던 유사 기능 프로그램의 대체
- **개발 목표**
  - 직관적이고 빠른 UI/UX
  - 규칙 기반 이름 변경 기능
  - 드래그&드롭으로 목록 순서 변경
  - 미리보기 기능
  - Undo
  - 대용량·다량 파일 안정적 처리
  - 무설치, 포터블 구조 배포

# 2. 핵심 기능

## 파일 추가

- 탐색기 열기, 드래그&드롭 방식 모두 지원

## 파일 목록

표 구성: **번호 | 현재 이름 | 변경될 이름 | 크기 | 경로**

- 번호는 파일만 순차적으로 표시
- 폴더는 번호 대신 빈칸 표시(번호 카운트에서 제외)

## 이름 변경

- 기본 제공 tag + custom tag 제공
- tag와 고정 텍스트를 조합한 형식 작성
  - **예시:** `[tag1] tag2_tag3 (tag4)`
- 규칙 적용 시, 리스트 순서 1부터 일괄적용
- custom tag 및 작성한 규칙은 프로그램 종료 전까지 유지됨

## 리스트 정렬

- 기본 정렬 기준: 파일 추가 순
- 정렬 요소:
  1. 파일 추가 순
  2. 파일 이름
  3. 파일 생성일
  4. 파일 수정일
  5. 파일 크기
- 드롭다운 + 오름차순/내림차순 버튼 제공

## 확장자 수정

- 확장자 표시 ON/OFF 토글
- 확장자 변경 기능 제공
